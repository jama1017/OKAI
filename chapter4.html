Chapter 4

Perceptron Learning

How we make a perceptron to learn to perform certain tasks? In this chapter, we are going behind the scenes of “Perceptron Learning Institute”.

Scroll

[Change wording for transition] Now that we have a perceptron that can take in some input and spit out a valid output, how can we make the perceptron learn? To understand this, we must first discuss what a perceptron is [abstractly].

Perceptrons can be seen as a mathematical function, taking in some inputs, some weights, a bias, and mixes them up mathematically to generate an output.

Since we have no control over the inputs (we can’t change the data we are learning from), we will try to change weights and balances so that the perceptron function will give out a good output for all of the input we stuff into it.

The core of “teaching” a perceptron is to let it understand how far its actions or predictions are from the ground truth or optimum. To achieve this, we use loss functions to measure the performance of our model. A commonly used loss function for classifying images is called cross entropy.

We want the model to be as accurate as possible, so in other words, we want the cross entropy loss to be as close to 0 as possible. In other words, we want our predictions to be as close to reality as possible. To achieve this, the loss of the model would decrease as we train the model over and over again.

There are many other loss functions, but all loss functions share a common behavior: the further off the prediction is from the ground truth, the bigger the output of the loss function is. This way, we can try to minimize the loss by changing the weights and biases, and the model will gradually improve until it reaches its capacity.

[Change this to go with animation more]To minimize loss, we use an optimizer called Gradient Descent, which, you can think of as walking down the loss function until all points around it are bigger or has the same value. . To walk “downward” you would find the steepest direction, and take a step in that direction, until you have slid all the way from high up on a hill to the lowest point in a valley.
