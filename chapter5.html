Chapter 5

Introduction to Feed-forward Neural Networks

Building upon perceptrons, Feed-forward Neural Networks (FNN) can complete far more powerful tasks. This chapter presents an anatomy of this simplest type of artificial nerual networks.

Scroll

You may have noticed, to this point, we have only discussed models with a single perceptron. These perceptron algorithms are not particularly powerful, as they are slow to learn, and cannot learn complex concepts. To learn more complex concepts, we need to stack perceptrons together so that they can form a more complex model.

In a graph of a network, the first layer is known as the input layer, and all the layers in the middle are called hidden layers.

The last layer is called an output layer. We say that a network has n layers when there are n layers that perform calculations, that is, there are n-1 hidden layers and 1 output layer.

[+ Given the temp range of today as input]This network is a simple two-layer perceptron network where the 2 inputs

are split into 4 perceptrons in the first layer, and then the outputs of the first layer is is fed into the 2 perceptrons in the second layer.

In this structure, all of the nodes in the first layer has a ReLU activation function, while the second is a linear layer (Meaning that it has no activation function). So why do we need activation functions when we don’t need to control the range of output? And what exactly is a ReLU function?

It turns out, that if we have consecutive linear layers, the network will behave exactly the same as a network with a single linear layer. To make the extra layer count, we need to introduce non-linearity. If you are interested in the mathematical reason behind this, this is a good artical to read. One of the most popular activation functions to break linearity is ReLU. ReLU stands for rectified Linear Unit, and maintains the value if the input is bigger than 0, and if the input is negative, it forces everything to be zero. In other words, it takes the maximum of 0 and the input.

With these activation functions, we can stack more layers together so they can learn to perform more complex tasks, for example recognizing numbers as showcased in the introduction.
